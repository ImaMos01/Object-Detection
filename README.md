## Setup

### FastAPI

- `virtualenv yoloProy`

- `source yoloProy/bin/activate`

- `python -m pip install -r requirements.txt`

#### run

- `cd objectDetectionAPI`

- `uvicorn main:app --reload`

### Web Page (React)

- `cd frontend`

- `npm run dev`
